TravelXP, created by Akshath and Sarvesh

TravelXP is an Android application that allows users to log, view, and manage their travel experiences. Users can create trips with destinations, dates, ratings, companions, and notes, which are securely stored using Firebase Firestore. The app supports full CRUD functionality, lifecycle-aware draft saving using SharedPreferences, and multi-activity navigation with a modern RecyclerView-based trip list. Authentication is handled through Firebase Email/Password login. The UI follows a clean, modern design inspired by the Indiana University Hoosiers cream and crimson color scheme, featuring rounded buttons and image-enhanced layouts. TravelXP demonstrates core Android concepts including lifecycle management, persistent storage, and backend integration.
